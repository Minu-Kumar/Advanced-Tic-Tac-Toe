# Advanced-Tic-Tac-Toe
Advanced Tic Tac Toe is a Python-based desktop game featuring AI opponents (Easy, Medium, Hard), score management, player dashboard, persistent data storage using JSON/CSV, sound effects, and a user-friendly Tkinter interface.


A Python Tkinter-based Advanced Tic Tac Toe game with:

## Features
- Easy, Medium, and Hard AI levels
- Minimax Algorithm for Hard mode
- Player name system
- Score tracking (Wins, Losses, Draws)
- Dashboard with statistics
- Sound effects on losing
- Modern GUI using Tkinter

## Technologies Used
- Python
- Tkinter
- Pygame
- JSON
- CSV
